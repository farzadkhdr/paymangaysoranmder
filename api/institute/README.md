# APIی سیستەمی پەیمانگای سۆران

ئەم APIیە بۆ یەکلاکردنەوەی سیستەمی مامۆستا و سیستەمی پەیمانگا دروست کراوە.

## دەستپێکردن

### پێویستییەکان
- Node.js 16+ 
- npm

### جێبەجێکردن

```bash
# Clone کردنی پڕۆژەکە
git clone [repository-url]

# چوونە ناو فۆلدەرەکە
cd api/institute

# دامەزراندنی پاکەتەکان
npm install

# دەستپێکردنی سێرڤەر
npm start

# بۆ پەرەپێدان
npm run dev
